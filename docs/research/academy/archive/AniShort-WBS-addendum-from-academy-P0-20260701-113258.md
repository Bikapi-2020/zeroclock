## 1. 对齐原则

本增量不替换主 WBS，而是把 Academy P0 视频确认的真实工作流拆成可开发、可测试、可验收任务。后续开发时：

- 每个任务都必须能映射到 PRD 条目。
- PRD 或验收描述不清晰时，必须使用 Chrome/CDP 回到原站核验，不能自行猜测。
- 视频证据只能作为开发依据之一；涉及按钮启用、接口参数、错误提示、模型限制时，还要查原站交互或前端资源。

## 2. Phase 对齐总表

| 主 WBS Phase | Academy P0 新增任务 | 验收重点 |
| --- | --- | --- |
| Phase 1 平台基础 | Academy 教程中心、个人/团队空间差异 | 教程可播放，团队/个人空间隔离 |
| Phase 2 画布内核 | 右键菜单、节点搜索、历史版本入口、全屏查看 | 画布基础操作与原站一致 |
| Phase 3 节点系统骨架 | 节点动作菜单、端口类型、节点分组、批量操作入口 | 节点库可覆盖 P0 工作流 |
| Phase 4 剧本/主体/分镜 | 智能分段、固定分段、主体库状态、Scene 分镜结构 | 剧本到分镜可完整跑通 |
| Phase 5 模型网关 | 图像/视频模型列表、推荐模型、模式限制 | 模型选择、任务状态、失败返还 |
| Phase 6 风格一致性 | 主体库、720 全景、全局风格提示词 | 角色/场景/道具一致性 |
| Phase 7 批量生产 | 批量生成分镜图、批量视频、整组执行 | 批量任务可排队和重试 |
| Phase 8 AI 助手 | AI 改写、结果回填剧本节点 | 助手输出可进入节点 |
| Phase 9 3D 导演台 | 独立 3D 工作台、角色白模、场景比例/站位 | 3D 输出可回到图像/视频链路 |
| Phase 12 协作计费后台 | 团队充值、成员积分、模型权限、生成统计 | 团队生产闭环可管理 |

## 3. Phase 1 增量：Academy 与团队/个人基础

### 3.1 任务

- 建立 Academy 教程列表页。
- 建立教程详情页与视频播放器。
- 建立教程分类管理。
- 建立教程推荐/置顶字段。
- 建立个人空间与团队空间容量字段。
- 建立个人积分与团队积分账户。
- 建立团队成员积分分配记录。

### 3.2 数据表补充

- `academy_categories`
- `academy_tutorials`
- `credit_accounts`
- `credit_allocations`
- `workspace_storage_quotas`

### 3.3 测试

- 单元测试：教程分类映射、视频 URL CDN 转换、空间容量计算。
- 集成测试：创建教程、发布教程、按分类查询教程。
- E2E：进入 Academy -> 筛选分类 -> 播放视频 -> 返回列表。
- E2E：团队创建者充值 -> 给成员分配积分 -> 成员生成任务消耗团队积分。

### 3.4 验收标准

- Academy 至少能展示入门、全流程教程、基础教程、功能教程、新功能、常见问题分类。
- 教程视频可播放，封面、时长、标题、描述一致。
- 个人项目与团队项目空间隔离。
- 团队成员只能使用已分配或团队授权积分。

## 4. Phase 2 增量：画布工作区真实操作

### 4.1 任务

- 增加画布历史版本入口。
- 增加节点搜索与定位。
- 增加右键菜单。
- 增加全屏查看。
- 增加底部快捷工具条：缩放、选择/移动、适应视图、对齐、添加节点、主体库、资源备份、全局风格。
- 增加画布保存状态：已同步、保存中、失败。

### 4.2 测试

- 单元测试：画布操作 reducer。
- 集成测试：创建节点、连线、保存、刷新恢复。
- E2E：右键创建节点 -> 连线 -> 保存 -> 刷新 -> 节点仍在。
- E2E：删除节点 -> 历史版本回退 -> 节点恢复。

### 4.3 验收标准

- 空白区域右键可创建节点。
- 选中节点可全屏查看和打开参数面板。
- 保存状态准确反映本地和服务端状态。
- 历史版本至少支持查看和回退。

## 6. Phase 4 增量：剧本、主体、分镜

### 6.1 任务：剧本分段/优化

- 实现智能分段。
- 实现固定分段。
- 实现 1-15 秒段落约束。
- 实现剧本整理。
- 实现情节优化。
- 实现人物/场景/道具提示词生成。
- 实现输出回填剧本节点。

### 6.2 任务：主体提取/生成

- 实现角色、场景、道具提取。
- 实现主体库状态匹配。
- 实现同名/别名归并。
- 实现角色多视图生成。
- 实现场景设定图生成。
- 实现道具三视图生成。
- 实现主体命名与入库。
- 实现音色主体绑定接口预留。

### 6.3 任务：分镜生成

- 实现 Scene 结构化输出。
- 实现四宫格/九宫格推导。
- 实现分镜资产汇总表。
- 实现图像宫格提示词。
- 实现视频提示词。
- 实现旁白/音画同步/负面提示词。
- 实现批量生成分镜图入口。

### 6.4 测试

- 单元测试：智能分段时长约束。
- 单元测试：别名合并。
- 单元测试：Scene 时长到四宫格/九宫格映射。
- 集成测试：剧本 -> 主体提取 -> 主体生成。
- 集成测试：剧本分段 -> 分镜生成 -> Scene 输出校验。
- E2E：输入短剧剧本 -> 生成主体表 -> 生成角色设定图 -> 入库 -> 生成分镜。

### 6.5 验收标准

- 分段结果每段时长在规则范围内，固定分段可控。
- 主体提取结果有角色、场景、道具分类和主体库状态。
- 主体生成结果必须可以命名并加入主体库。
- 分镜输出必须包含 Scene 表、宫格图提示词、视频提示词、旁白、负面提示词。
- 生成失败时不丢失已有输出，支持重试。

## 7. Phase 5 增量：模型与任务限制

### 7.1 图像模型任务

必须配置：

- Banana Pro。
- Banana 2.0。
- Seedream 5.0。
- Seedream 4.5。
- Midjourney-v7/v8。
- GPT-image2 low/medium/high。

测试：

- 模型配置可启停。
- 默认模型可按项目/节点覆盖。
- 积分预估按模型和生成数量变化。

验收：

- 图像节点能选择所有启用模型。
- 推荐模型默认展示 Banana Pro。
- 失败任务积分返还或标记待返还。

### 7.2 视频模型任务

必须配置：

- VEO 3.1 / Pro。
- Seedance 1.5。
- Seedance 2.0 / fast / mini。
- 可灵 2.6 / V3 / V3 Omini / 动作迁移。
- Sora 2。
- Vidu Q3 Pro / Turbo。
- Grok Video 3。
- Wan 2.7。
- HappyHorse 1.0。

测试：

- 首尾帧模式缺首帧时阻止生成。
- 视频编辑模式无参考视频时阻止生成。
- 动作迁移无参考图或参考视频时阻止生成。
- 参考视频时长超限时阻止生成。
- 批量生成时每个任务有独立状态。

验收：

- Seedance 2.0 作为短剧主流程推荐模型。
- 不同 referenceMode 会正确组织 images/video_url/voice_url。
- SSE/轮询中断不会错误标记为失败。

## 8. Phase 6 增量：一致性、主体库与 720 全景

### 8.1 任务

- 主体库支持角色、场景、道具、音色。
- 主体库支持跨项目导入。
- 主体库支持多状态/妆造。
- 图像节点支持全景生成。
- 720 全景可标记为场景背景。
- 全局风格支持自定义提示词和风格节点。

### 8.2 测试

- 单元测试：主体库查重。
- 单元测试：主体引用替换。
- 集成测试：图像 -> 全景 -> 3D 背景。
- E2E：场景设定图 -> 生成 720 全景 -> 进入 3D 导演台作为背景。

### 8.3 验收标准

- 同一角色在多节点可被稳定引用。
- 全景图生成后可全屏查看并作为 3D 背景。
- 全局风格变更后，下游节点明确提示需刷新或重新生成。

### 9.1 任务

- 整组执行。

- 批量替换素材。

### 9.2 测试

## 10. Phase 8 增量：AI 助手与回填

### 10.1 任务

- 助手读取画布上下文。
- 助手回答剧本/提示词问题。
- 助手输出可复制或回填指定节点。
- 助手只有用户明确要求改画布时才生成操作卡片。

### 10.2 测试

- 单元测试：上下文裁剪。
- 集成测试：助手输出回填剧本节点。
- E2E：让助手优化剧本 -> 拷贝到剧本节点 -> 执行分段。

### 10.3 验收标准

- 助手不会未经确认直接改画布。
- 回填后触发保存并记录历史版本。

## 11. Phase 9 增量：3D 导演台

### 11.1 任务

- 独立 3D 导演台页面/全屏编辑器。
- Three.js 视口。
- 加载 720 全景/场景背景。
- 添加人物白模。
- 拖拽角色到场景。
- 调整角色比例、位置、旋转。
- 调整相机视角。
- 保存多个镜头。
- 输出参考图回写画布。

### 11.2 测试

- 单元测试：3D 场景数据序列化。
- 集成测试：导演台保存/恢复。
- Playwright + 截图：打开 3D 导演台，场景非空白。
- E2E：全景图 -> 3D 导演台 -> 添加角色 -> 输出参考图 -> 图像节点生成。

### 11.3 验收标准

- 角色白模可拖入场景并使用 TransformControls 调整。
- 场景比例和人物比例可保存并恢复。
- 输出参考图能进入画布节点链路。
- 移动端可不作为 P0，但桌面端必须稳定。

## 13. 新增核心 E2E 验收路径

### 13.1 Academy 教程路径

1. 进入 Academy。
2. 切换到全流程教程。
3. 打开基础短剧工作流。
4. 视频可播放。
5. 返回列表。

### 13.2 基础短剧工作流路径

1. 创建项目。
2. 打开基础短剧模板。
3. 输入剧本。
4. 运行剧本分段。
5. 运行主体提取。
6. 生成一个角色主体并入库。
7. 生成一个 Scene 分镜。
8. 生成一张分镜图。
9. 生成一个视频。

### 13.3 自由式工作流路径

1. 新建空白画布。
2. 右键创建剧本节点、主体生成、图像生成、视频生成。
3. 手动连线。
4. 上传参考图。
5. 局部执行图像生成。
6. 局部执行视频生成。

### 13.4 720 全景 + 3D 路径

1. 图像节点生成场景图。
2. 右键选择全景生成。
3. 生成 720 全景图。
4. 打开 3D 导演台。
5. 设置全景为背景。
6. 拖入人物白模。
7. 调整比例和站位。
8. 输出参考图。

### 13.5 团队模式路径

1. 创建团队。
2. 团队充值。
3. 给成员分配积分。
4. 创建团队项目。
5. 成员进入项目并生成任务。
6. 管理员查看成员生成记录和积分消耗。

## 14. 下一批视频处理建议

P1 视频应优先处理：

1. 画面风格节点。
2. 图像节点打光生成。
3. 多节点批量操作。
4. 输出高度可控的参考图，告别视频抽卡。
5. 截取视频首尾帧、任意帧。
6. 生成好的设定图如何替换。
7. 一个角色多个名字怎么办。
8. 一个角色多种状态（妆造）。
9. 如何让人物音色始终不变。
10. 自动保存-历史版本。
